# 🚀 iOS 快捷指令集 (Shortcuts)

欢迎尝试我的快捷指令仓库！本仓库的快捷指令工作模式（提供数据--->跳转本人网站展示结果）

---

## 快捷指令合集

### [影视Play](https://www.icloud.com/shortcuts/9bce53ce341641c1a8c24930d13e0221)
> **简介**：输入影视名称，从全网开放接口中搜索汇总到一起。
> ![影视Play预览图](https://github.com/SheepFJ/Shortcuts/blob/main/Shortcuts/Effect/%E5%BD%B1%E8%A7%86Play%E6%95%88%E6%9E%9C.png)

---

### [短视频解析](https://www.icloud.com/shortcuts/ea84365c938b4e33afa1681242cc9718)
> **简介**：支持抖音，b站等平台短视频去水印解析，如有不支持的请加群反馈。
> ![短视频解析预览图](https://github.com/SheepFJ/Shortcuts/blob/main/Shortcuts/Effect/%E7%9F%AD%E8%A7%86%E9%A2%91%E8%A7%A3%E6%9E%90%E6%95%88%E6%9E%9C.png)

---

### [音乐播放器](https://www.icloud.com/shortcuts/e7382ad6740e4d54b351a72cc0fed7ec)
> **简介**：歌曲来源于酷狗音乐，输入歌曲名/作者，跳转到网页播放。
> ![音乐播放器预览图](https://github.com/SheepFJ/Shortcuts/blob/main/Shortcuts/Effect/%E9%9F%B3%E4%B9%90%E6%92%AD%E6%94%BE%E5%99%A8%E6%95%88%E6%9E%9C.png)
